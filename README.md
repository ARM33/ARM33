## 🚀 Featured Projects

### 🧬 Genomic Digital Twin Platform *(Private — Research)*
> **Python · PostgreSQL · KEGG REST API · Web Scraping · XML/KGML · Graph Analysis**

End-to-end bioinformatics data platform for large-scale genomic analysis of *E. coli*. Integrated 8+ biological databases (KEGG, EcoCyc, SABIOrk) into a unified PostgreSQL schema. Built automated data ingestion pipelines combining REST APIs and web scraping for DNA sequence, metabolic pathway, and reaction data. Developed graph-based visualization to reconstruct metabolic networks across 100+ pathways.

*Code is under research confidentiality agreement — architecture documentation available on request.*

### 🌱 MicrovitroLab — Agrobiotechnology Traceability System
> **Angular 20 · TypeScript · PrimeNG · Tailwind CSS · REST API · Node.js**

[![Repo](https://img.shields.io/badge/Repo-AngularPractices-181717?style=flat&logo=github)](https://github.com/ARM33/AngularPractices)

Full-stack traceability platform for plant tissue culture production, tracking 6-stage workflows (E0→E5) from GPS-based field collection to greenhouse delivery. Reactive Angular 20 architecture using Signals and the Vista–Store–Service pattern. Modules: lot management, project tracking, client registry, production stage engine, and KPI dashboards.
